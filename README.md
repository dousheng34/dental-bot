# 🦷 DENT PLUS — Telegram Bot + Mini App для стоматологии

## 📁 Структура проекта
```
dental_bot/
├── app/
│   ├── run.py           ← Запуск всего (бот + сервер + планировщик)
│   ├── bot.py           ← Telegram бот (aiogram 3)
│   ├── main.py          ← FastAPI сервер + REST API
│   ├── database.py      ← База данных SQLite (10 таблиц)
│   ├── scheduler.py     ← Автонапоминания пациентам
│   └── static/miniapp/
│       └── index.html   ← Mini App (запись, зубная карта, история)
├── requirements.txt
├── Dockerfile
├── docker-compose.yml
└── .env.example
```

---

## ⚙️ БЫСТРЫЙ СТАРТ

### Шаг 1 — Получи токен бота
1. Открой [@BotFather](https://t.me/BotFather)
2. `/newbot` → введи имя → скопируй токен

### Шаг 2 — Узнай свой Telegram ID
1. Открой [@userinfobot](https://t.me/userinfobot)
2. Нажми `/start` → скопируй ID

### Шаг 3 — Настрой .env
```bash
cp .env.example .env
```
Открой `.env` и заполни:
```
BOT_TOKEN=1234567890:AAF...твой_токен
ADMIN_ID=123456789
WEBAPP_URL=https://твой-домен.railway.app
```

### Шаг 4 — Установка зависимостей
```bash
pip install -r requirements.txt
```

### Шаг 5 — Запуск
```bash
cd app
python run.py
```

### Шаг 6 — Деплой на Railway (бесплатно)
1. [railway.app](https://railway.app) → New Project → Deploy from GitHub
2. Settings → Variables → добавь BOT_TOKEN, ADMIN_ID, WEBAPP_URL
3. Railway автоматически запустит через Dockerfile ✅

### Шаг 7 — Подключи Mini App к боту
1. [@BotFather](https://t.me/BotFather) → `/newapp`
2. Выбери своего бота
3. URL: `https://твой-домен.railway.app`

## 🎯 ПОЛНЫЙ СПИСОК ФУНКЦИЙ

### 👤 ПАЦИЕНТ (Mini App — 4 вкладки)
| Вкладка | Функции |
|---------|---------|
| 📅 Запись | Выбор услуги по категориям, врача, даты, времени. Комментарий, телефон. Оплата Kaspi |
| 🦷 Зубная карта | Интерактивная карта 32 зубов. Статусы: здоровый/кариес/пролечен/удалён/коронка/имплант |
| 📋 Мои записи | Все записи с статусами. Отзыв после приёма |
| 👤 Профиль | Статистика, баллы лояльности, форма отзыва |

### 👨‍⚕️ ВРАЧ (Telegram Bot)
| Кнопка | Функция |
|--------|---------|
| 📅 Расписание сегодня | Все пациенты на день: имя, услуга, телефон |
| 📅 Завтра | Предварительный список |
| 📊 Статистика | Заработок за месяц, рейтинг по отзывам |

### 👩‍💼 АДМИНИСТРАТОР (Telegram Bot)
| Кнопка | Функция |
|--------|---------|
| 📅 Записи сегодня | + кнопки: ✅ Подтвердить, ❌ Отменить, ✔️ Выполнено |
| 🔬 Лаборатория | Статусы заказов + отметить "Готово" |
| 💰 Финансы | Выручка сегодня / месяц, долги |
| 📊 Статистика | Пациенты, записи, выручка |
| 🔗 Ссылка | Ссылка для пациентов (в Instagram, 2GIS) |

### ⏰ АВТОНАПОМИНАНИЯ (Планировщик)
| Время | Событие |
|-------|---------|
| 18:00 каждый день | 📨 Напоминание за 24ч до приёма |
| Каждые 30 мин | 🔔 Напоминание за 2ч до приёма |
| 09:00 каждый день | 🎂 Поздравление с ДР + скидка 10% |
| 10:00 каждый день | 💌 Напоминание вернуться (через 6 мес) |
| Каждые 2 часа | 🔬 Уведомление врачу о готовом лаб. заказе |

## 🗄️ БАЗА ДАННЫХ (10 таблиц)
| Таблица | Данные |
|---------|--------|
| `clinics` | Название, адрес, телефон, Kaspi номер |
| `doctors` | ФИО, специализация, опыт, % зарплаты, Telegram ID |
| `service_categories` | Категории: Терапия, Хирургия, Ортодонтия... |
| `services` | 14+ услуг с ценами от/до и длительностью |
| `patients` | ФИО, телефон, аллергии, группа крови, баллы |
| `appointments` | Записи (врач, услуга, дата, время, статус) |
| `dental_chart` | Состояние каждого из 32 зубов |
| `treatments` | История лечения с привязкой к зубу |
| `invoices` | Счета, оплата, долги |
| `lab_orders` | Лабораторные заказы (коронки, протезы...) |
| `reviews` | Отзывы и рейтинги врачей |

## 🔧 API ENDPOINTS

| Метод | URL | Описание |
|-------|-----|----------|
| GET | `/api/categories/{clinic_id}` | Категории услуг |
| GET | `/api/services/{clinic_id}` | Список услуг |
| GET | `/api/doctors/{clinic_id}` | Врачи с рейтингом |
| GET | `/api/slots/{doctor_id}/{date}` | Свободные слоты |
| POST | `/api/appointment` | Создать запись |
| GET | `/api/patient/appointments/{id}` | Записи пациента |
| GET | `/api/patient/chart/{id}` | Зубная карта |
| POST | `/api/patient/chart` | Обновить зуб |
| GET | `/api/patient/treatments/{id}` | История лечения |
| POST | `/api/review` | Оставить отзыв |
